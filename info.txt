Installation
------------
The game needs 35Kb, a floppy disk and a joystick. In Vice make sure to enable all banks in the Machine settings.

Tools
-----
* Programming Language/IDE: Turbo Rascal Syntax Error: Great Pascal like compiler for 8-bit machines (www.turborascal.com)
* cc1541 (https://csdb.dk/release/?id=220661)
